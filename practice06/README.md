# Practice 06

This folder contains practical exercises for:
- File handling
- Directory management
- Built-in functions

## Structure

```text
practice06/
├── file_handing/
│   ├── read_files.py
│   ├── write_files.py
│   └── copy_delete_files.py
├── directory_management/
│   ├── create_list_dirs.py
│   └── move_files.py
└── builtin_functions/
    ├── map_filter_reduce.py
    └── enumerate_zip_examples.py
```

## What Each Part Does

### 1) file_handing
- `write_files.py`: creates/writes data into `sample.txt` and appends a new line.
- `read_files.py`: reads file content and prints it.
- `copy_delete_files.py`: practices copy/backup and safe delete logic.

### 2) directory_management
- `create_list_dirs.py`: creates nested directories and prints directory tree.
- `move_files.py`: creates test files, copies one file, and moves another.

### 3) builtin_functions
- `map_filter_reduce.py`: uses `map()`, `filter()`, and `reduce()`.
- `enumerate_zip_examples.py`: uses `enumerate()` and `zip()`, and demonstrates type checking/conversions.
