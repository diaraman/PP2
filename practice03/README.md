# Practice 03

This folder contains Python examples for functions, lambda, classes, and inheritance.

## Sections
- functions
- lambda
- classes
- inheritance

## File goals
- functions: basic function usage, arguments, return values, args and kwargs.
- lambda: lambda basics and usage with map, filter, sorted.
- classes: class definition, __init__, class methods, class variables.
- inheritance: basic inheritance, super(), overriding, multiple inheritance.

## Run examples
Use Python to run any file, for example:

python practice03/functions/basic_functions.py
python practice03/lambda/lambda_with_map.py
python practice03/classes/init_method.py
python practice03/inheritance/super_function.py
