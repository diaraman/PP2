# Practice 04

This folder contains beginner Python exercises on:
- generators and iterators
- date and time
- math operations
- JSON basics

## Files

- `generators.py`  
  Simple iterator and generator examples.

- `dates.py`  
  Basic datetime examples.

- `mathh.py`  
  Simple math/random examples.

- `jsonn.py`  
  Basic JSON conversion examples.

## Exercises

Inside `exercises/`:
- `math.md` - all math exercise solutions
- `generator.md` - all generator exercise solutions
- `date.md` - all date exercise solutions
