# Moving Ball Game

## Features
- White background
- Red ball (`radius = 25`, so size is `50x50`)
- Arrow keys move ball by `20` pixels
- Ball cannot leave screen boundaries
- Invalid moves (off-screen) are ignored

## Controls
- `Left Arrow` - move left
- `Right Arrow` - move right
- `Up Arrow` - move up
- `Down Arrow` - move down

## Run
```bash
cd Practice7/moving_ball
python3 main.py
```
