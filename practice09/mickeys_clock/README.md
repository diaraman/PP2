# Mickey's Clock

## Objective
A simple Mickey-style clock made with `pygame`.

## Features
- Shows current system time (minutes and seconds)
- Rotates Mickey hands in real time
- Updates smoothly in a game loop

## Project Structure
```
mickeys_clock/
├── main.py
├── clock.py
└── images/
    ├── clock_bg.png
    └── mickey_hand.png
```

## Run
```bash
cd practice09
python3 mickeys_clock/main.py
```

## Notes
- `main.py` handles window + loop
- `clock.py` contains clock drawing logic
